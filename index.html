Checkout our Recent Events

CodeJam-01

On November 5, 2023, the CCET ACM and ACM-W Student Chapters collaboratively hosted an event called 'CodeJam 101' . In this event the participants were offered an excellent opportunity to learn the essential things in coding and familiarize themselves with the programming languages as well as how to code. The speakers, Vanshika Bhardwaj, Saksham Arora and Simran Jaggi, shared information and provided hands-on guidance, making this session an invaluable experience for all participants where they were able to learn coding along with trying it.

Write it Right

On November 4, 2023, the CCET ACM and ACM-W Student Chapters collaboratively hosted an Introductory Session on Technical Writing-Write it right. This event offered an excellent opportunity for attendees to familiarize themselves with the significance of technical writing and how to navigate various tools like Google Scholar and Kaggle. The speaker, Aishita Sharma, shared valuable insights and provided guidance, making this session an invaluable experience for all participants.

Introductory Session on Designing

On October 25, 2023, the CCET ACM and ACM-W Student Chapters collaboratively hosted an Introductory Session on Designing. This event offered an excellent opportunity for attendees to familiarize themselves with essential design tools such as Adobe Illustrator and Canva. The speaker, Japan Ajit Singh, shared valuable insights and provided hands-on guidance, making this session an invaluable experience for all participants.
